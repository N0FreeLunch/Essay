## add
- R.add()
- 인자로 받은 두 개의 값을 더하는 함수

## 설명
- 연산자 + 를 사용한 덧셈을 사용하는 방식은 변수에 값을 누적하는 방식이다.
- 함수형 프로그래밍은 변수가 아닌 함수에 값을 누적해서 합성된 함수를 나중에 지연평가하기 위한 것이기 때문에 add 함수를 사용하는 것이 순수주의적인 함수형 프로그래밍이 된다.

## 표현
```
Number → Number → Number
```
- 첫 번째 인자로 수를 받고
- 두 번째 인자로 수를 받고
- 함수를 평가할 때도 수를 반환

## 예제
```
R.add(2, 3);       //=>  5
R.add(7)(10);      //=> 17
```
