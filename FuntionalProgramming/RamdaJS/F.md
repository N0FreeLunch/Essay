## F
> A function that always returns false. Any passed in parameters are ignored
- 항상 거짓을 반환하는 함수이다. 파라메터로 전달된 어느 것도 무시된다.

## 표현
```
* → Boolean
```
- `* →` 임의의 인자를 받는다.
- `→ Boolean` 항상 false를 반환한다.

## 설명
- 어떠한 인자를 넣든지 항상 fasle를 반환하는 함수

## 예제
```
R.F(); //=> false
```
