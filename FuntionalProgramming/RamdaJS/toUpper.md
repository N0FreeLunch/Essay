## toUpper
> The upper case version of a string.
- 주어진 문자열의 대문자 표기 버전이다.
> See also toLower.

### 설명
- 모든 문자가 대문자 표기가 된 문자열을 반환한다.

### 표현
```
String → String
```
- `String →`: 첫 번째 인자로 문자열을 받고
- `→ String`: 각 문자가 대문자로 변환된 문자열을 반환한다.

### 예제
```js
R.toUpper('abc'); //=> 'ABC'
```
- 설명은 생략한다.
